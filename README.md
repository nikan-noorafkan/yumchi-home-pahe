# Yumchi Home Page

دموی لندینگ تعاملی Yumchi با Next.js و Framer Motion.

## اجرا

```bash
pnpm install
pnpm dev
```

سپس:
- `http://localhost:3000`
- `http://localhost:3000/fa`
